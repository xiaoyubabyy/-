## 创建（笔记）库的流程

- 创建一个文件夹
- 建立一个后缀为.md的文件
- git init - 初始化
- git add .
- git commit -m""
- 在github中建立一个库，复制其中一句代码
- git push
- 根据提示再粘贴一句代码
- git push
- 创建成功
